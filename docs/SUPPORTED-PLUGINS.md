# Supported Plugins
This is the list of plugins that are currently supported by the colorscheme.

If there is the need for one that is not listed here, please open an issue or a pull request.

- Treesitter
- LSP Diagnostics
- Telescope
- nvim-tree
- nvim-cmp
- Gitsigns
- Neogit
- Fugitive
- Lualine
- Mason
- Lazy.nvim
- Which-key
- Hop
- Diffview
- nvim-notify
- Copilot
